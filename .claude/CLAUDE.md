# Rondas

AI-powered bill-splitting iOS app for the Colombian market, built with Expo + React Native.

Be concise in all responses.

## Package Manager

pnpm

## Commands

```bash
pnpm dev        # start Expo dev server
pnpm typecheck  # tsc --noEmit
pnpm lint       # expo lint
```
